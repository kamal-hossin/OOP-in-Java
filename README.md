# OOP in Java — Example Programs

This repository contains small Java examples demonstrating core Object-Oriented Programming (OOP) concepts.

## Overview
- Each `.java` file illustrates a single OOP concept or related examples.

## Files and concepts
- `abstraction.java` — demonstrates abstraction (abstract classes/methods).
- `class_object.java` — shows how to define classes and create objects.
- `constructor.java` — illustrates constructors and initialization (possibly overloading).
- `encapsulation.java` — uses access modifiers with getters/setters to encapsulate state.
- `hierarchical_inheritance.java` — hierarchical inheritance: one base class with multiple subclasses.
- `hybrid_inheritance.java` — hybrid-style inheritance using interfaces and classes (Java doesn't support multiple class inheritance).
- `inner_class.java` — examples of inner and nested classes.
- `interface_in_java.java` — defines and implements interfaces.
- `multi_level_inheritence.java` — multi-level inheritance chain (A -> B -> C).
- `polymorphism.java` — method overloading and overriding (compile-time and runtime polymorphism).
- `single_inheritence.java` — single inheritance example (subclass extends superclass).
- `supur_class.java` — demonstrates usage of a superclass (likely a typo for `super_class`).
- `user_input.java` — reading input from the user (e.g., using `Scanner`).

## How to compile and run
1. Open a terminal in this project folder.
2. Compile all sources:

```bash
javac *.java
```

3. Run the class that contains a `main` method, e.g.:

```bash
java class_object
```

Replace `class_object` with the class name that has the `main(String[] args)` method you want to execute.

## Requirements
- Java JDK 8 or newer installed and `javac`/`java` available on your PATH.

## Notes
- File names may contain minor typos (e.g., `multi_level_inheritence.java`, `supur_class.java`). You can rename them if desired.
- Some files may not include `main` methods; compile all and run the intended example class.
